# Setup

## Requirements:

* Python 2.7

## Installation:

* Create virtualenv
```
$ virtualenv env
$ source env/bin/activate
```
* Install dependencies
* You are ready!

To deactivate virtualenv just type:
```
$ deactivate
```

## Run app:

```
python app.py
```
